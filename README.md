## Quick Start


```bash

# Install dependencies
npm install

# Run the React app (client)
npm run start

# Run test
npm run test

# App runs on http://localhost:3000




# Run build
#(It is included to the repo as it was mentioned in the requirements)
serve -s build

# App runs on http://localhost:5000
```
